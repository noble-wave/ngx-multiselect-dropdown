Quick Start

Get up and running in under 2 minutes

01

Install via npm

Add the package to your Angular project

bash 
npm install ngx-multiselect-dropdown
02

Import in your component

Use standalone component imports (Angular 21+)

typescript 
import { signal } from '@angular/core';
import { NgxMultiselectDropdownComponent } from 'ngx-multiselect-dropdown';

export class MyComponent {
  items = signal(['Apple', 'Banana', 'Orange']);
  selectedItems: string[] = [];
}
03

Use in your template

Add the dropdown to your HTML

html 
<ngx-multiselect-dropdown
  [data]="items()"
  [(ngModel)]="selectedItems"
/>

02Object Array with Field Mapping

Work with object arrays by configuring field mapping. Specify which fields to use for id and display text in the config object.

Live Demo

{{ citySelection().length }} cities

Component Class (app.ts)

cities = signal([{ id: 1, name: 'NY' }]);
selection = signal([]);
config = signal({
  fieldMapping: {
    idField: 'id',
    textField: 'name'
  }
});

Template (app.html)

<ngx-multiselect-dropdown
  [data]="cities()"
  [config]="config()"
  [(ngModel)]="selection"
/>

Configuration API

Complete configuration reference with all available options

Property Type Default Description
singleSelection boolean false Enable single selection mode instead of multi-select
fieldMapping object {} Map object properties: idField, textField, disabledField
enableSelectAll boolean true Show "Select All" / "Unselect All" checkbox option
selectAllText string 'Select All' Custom text for "Select All" action
unselectAllText string 'Unselect All' Custom text for "Unselect All" action
enableSearch boolean false Enable search/filter functionality for items
searchPlaceholder string 'Search...' Placeholder text shown in search input field
clearSearchOnClose boolean true Clear search filter when dropdown closes
selectionLimit number -1 Maximum items that can be selected (-1 for unlimited)
displayLimit number 3 Max selected items to display before showing "+X more"
closeOnSelect boolean false Close dropdown automatically after each selection
showSelectedAtTop boolean false Display selected items at the top of the dropdown list
maxHeight number 300 Maximum height of dropdown list in pixels
noDataText string 'No items' Message displayed when no data is available
noResultsText string 'No results' Message displayed when search returns no results

Example Configuration

// Component Class (app.ts)
config = signal<DropdownConfig>({
  singleSelection: false,
  enableSelectAll: true,
  enableSearch: true,
  searchPlaceholder: 'Type to search...',
  selectionLimit: 5,
  displayLimit: 2,
  closeOnSelect: false,
  maxHeight: 250,
  fieldMapping: {
    idField: 'id',
    textField: 'name',
    disabledField: 'disabled'
  }
});

// Template (app.html)
<ngx-multiselect-dropdown
  [data]="items()"
  [config]="config()"
  [(ngModel)]="selection"
/>
